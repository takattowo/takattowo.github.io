As an effort to revive the dying game, takattoFS2 Patcher was made to easily download and apply stuffs (called DLCs or patches) into your FreeStyle 2 client, come with a lot of QOL tweaks that single handy make the game better!

Due to the launcher being consistenly checking all the changes made to the files, many patches and tweaks can not be applied permanently so you will have to keep the patcher running. It helps to automatically apply DLCs and the original files are backed up, they will be revivified after the client closes.

The stuffs are downloaded from my "work collections" stored on cloud services. Some files like translations, character voices,... are fetched from Korea and China server.

takattoFS2 Patcher requires the game FreeStyle 2 to be installed, all servers supported (China, Korea and Global working).

Many players been using it and they said it gives them a cool experience over the vanilla one, making them wanna stick to the game for a longer time. They also expressed a lot of loves to my work and I am very happy to hear that!

Frequently Asked Question

For any error when running the patcher, try these fixes first:
  • Make sure you have the requirements to install or install Microsoft .NET Framework 4.7.2~
  • Add takattoFS2 Patcher to your anti-virus exception list as they usually slow down the program.
  • Reset Internet Explorer settings, Internet Explorer advanced settings and Internet Explorer security zones. Those can be found in Control Panel -> Internet Options.
  • Disable Windows's custom scaling, as it tends to make program visually broken.
  • If you encountered Unhandled exception, please take a note or screenshoot and report to me (it's important!) so I can work on a fix! Also keep in mind the patcher works on Windows 10 version 1709+ and Windows 7 SP1+ only.


Is it safe? Will I get banned?

This is not a hack tool, it customizes the game like changing map, changing music and doing absolutely harmless things,... If you are looking for hacks/cheats then you should not be here. The patcher is safe to use as it doesn't cause any harm to the game and players thus impossible to detect. It is up to the game developers whether they want to patch this patcher or not. If they do then you can say the game is completely dead, please move on.


What is WIP/STD/OB?

WIP stands for "Work in process". STD stands for "Standalone", patches marked as standalone no need to patch everytime you run the game. OB stands for "Obsolete", patches marked as obselete are no longer working or need to do extra steps in order to make them work.
